# Personal Planner and Habit Tracking System

## Overview

A simple Personal Planner and Habit Tracking System that helps users organize daily tasks, track habits through streaks and pie chart analytics, and manage notes efficiently. No login is required, making it easy to use and accessible for everyone.

## Features

* Daily task planning
* Habit tracking with streak monitoring
* Pie chart progress visualization
* Notes management
* Local storage support
* Responsive and user-friendly interface
* No authentication required

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API
